# Rust TUI スライドへようこそ！

これはテスト用の1枚目のスライドです。

---

## 2枚目のスライド

- Rust
- 所有権

---

## 3枚目: コードブロックのテスト

Rustのコードはこんな感じに表示されます。

```rust
fn main() {
    println!("Hello, TUI!");
}
```

---

## 4枚目: 引用と太字のテスト

通常のテキストの中に、**ここだけ太字**のような装飾ができます。

> これは引用ブロックです。
> 大事な言葉や、外部からの引用を目立たせます。

- リストの中でも**太字**が使えます！
